# Codex 训练营 L3 优秀作业

## 项目

评审虾质量左移指标看板

## 启动方式

在本目录执行：

```bash
cd app
python3 -m http.server 4173
```

然后打开：

```text
http://127.0.0.1:4173
```

如果系统 Python 不可用，可使用 Codex runtime Python：

```bash
/Users/yangcong/.cache/codex-runtimes/codex-primary-runtime/dependencies/python/bin/python3 -m http.server 4173
```

## 文件结构

```text
app/                 可运行网页 App
docs/                作业报告、开发计划、验收报告
project-skills/      项目沉淀 Skills
```

## MVP

- 记录试点需求
- 自动计算 5 个质量左移指标
- 维护证据链和人工确认点
- 支持筛选、详情查看和周同步导出
- 沉淀项目 Skills
